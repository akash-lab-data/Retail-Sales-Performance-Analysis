# Retail-Sales-Performance-Analysis
Project overview
This project analyzes a retail sales dataset to identify key drivers of profitability, regional performance, and the impact of discount strategies. The goal is to uncover business insights that can help improve profitability and optimize pricing decisions.

Tools used
Excel – data cleaning and preprocessing
MySQL – business analysis using SQL queries
Python – exploratory data analysis and visualizations
Power BI – interactive executive dashboard

Project workflow
Data Cleaning (Excel)
Removed duplicates, fixed date formats, trimmed text fields, and created a profit margin column.

SQL Analysis (MySQL)
Performed business queries including sales by category, profit by region, loss percentage, and sub-category profitability.

Python EDA
Explored patterns in the data using pandas and visualizations to analyze category performance, discount impact, and sales trends.

Power BI Dashboard
Built an executive dashboard showing key KPIs, regional performance, category profitability, and discount impact.

Key insights
Technology has the highest profit margin among all categories.
Furniture generates the lowest overall profit.
Tables are the largest loss-making sub-category.
The Central region has the lowest profit performance.
18.7% of orders are loss-making.
Higher discount levels significantly reduce profitability.
Sales fluctuate month to month but show a clear upward yearly trend.

Business recommendations
Reduce excessive discounting on furniture products, especially tables.
Review pricing strategy for low-margin sub-categories.
Investigate operational performance in the Central region.
Focus marketing efforts on high-margin technology products.
